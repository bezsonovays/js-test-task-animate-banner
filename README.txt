Сделать анимированный рекламный баннер (display ad), который использует внешний источник данных, по приведенному макету.
https://www.figma.com/file/VvjysBwNDEgdMbgZTAcGaS/Nike-Test?node-id=74%3A4

В качестве источника данных используем json по ссылке httpssolovey.com.uatestdata.json
Баннер должен загружать json и использовать данные из него для отображения товарных предложений
 - Вся анимация - на вкус исполнителя.
 - После каждого показа позиции как-то анимировать кнопку “Order Now!”
 - Кол-во позиций в json - от 1, т.е. при наличии только одной позиции в json, анимация тоже должна проигрываться.
 - Клик по баннеру должен приводить к переходу по ссылке на отображаемую позицию (берем из того же из json).
 - По технологиям и либам - никаких ограничений (в разумных для баннера пределах), но, чем ванильнее (и, соответственно, легче) - тем лучше.
